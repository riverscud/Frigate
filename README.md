# Frigate Config and Notes

A collection of my Frigate NVR configuration files, setup notes, troubleshooting steps, and hardware-specific adjustments for self-hosted camera monitoring and object detection.

## Overview

This repository is used to organize and document my Frigate setup, including configuration files, detector settings, camera streams, retention settings, and integration notes. It also serves as a reference for troubleshooting and future changes.

## What’s in this repo

- Frigate configuration files
- Camera setup notes
- Stream and go2rtc settings
- Detector and hardware acceleration notes
- Storage and retention settings
- Integration notes for Home Assistant
- Troubleshooting and testing logs

## Goals

- Keep Frigate configs organized and version controlled
- Document changes before and after testing
- Track working camera and stream settings
- Save notes for future upgrades and troubleshooting
